JavaScript frameworks

Angular JS

Informatii generale

Probabil primul framework care a atins un grad de acoperire si utilizare mai mare decat jQuery. Pornit undeva in Google, se afla pe ultimele picioare, desi se mai gasesc aplicatii multe care folosesc Angular JS, devine problematica mentinerea iar suportul oficial al echipei de dezvoltare se va termina in curand

Printre avantajele folosirii Angular JS se numara:

  • Apropiat de JavaScript, nu necesita Typescript sau Babel
  • Framework complet, contine orice de la comunicare cu serverul la stockarea cookies
  • Functionalitati puternice de validat formulare

Dezavantajele folosirii Angular JS cuprind:

  • Concepte invechite greu de scalat si intretinut
  • Suportul urmeaza sa se termine
  • Se gasesc greu dezvoltarii dispusi sa migreze sau sa mai creeze functionalitati noi

Angular

Informatii generale

Angular este varianta curenta a fostului framework Angular JS. Desi pastreaza cateva din conceptele variantei initiale, aceasta este destul de diferita si mai dificil de invatat, pe de alta parte permite mai putine greseli din partea dezvoltatorilor, in special din cauza utilizarii TypeScript.

Printre avantajele folosirii Angular se numara:

  • Comunitate mare online in continua crestere
  • Nu exista risc major de abandonare a dezvoltarii ca la Anguar JS
  • Framework complet ce contine comunicarea cu serverul sau validare avansata de formulare

Dezavantajele folosirii Angular cuprind:

  • Dimensiuni mari ale aplicatiilor
  • Complexitate adaugata de TypeScript
  • Documentatia lasa de dorit in anumite sectiuni
